# Quickstart
A concise guide to quickly integrate Gateway into your workflow
Follow these steps:
Sign Up
  * **Sign Up:** Create an account on the [Gateway platform](https://gateway.sanctum.so/).
  * **Set Up Organization and Project:**
    * Create an **organization** to manage multiple projects.
    * Create **projects** within your organization to isolate and manage different use-cases, ensuring clear isolation, per-project analysis and optimized resource management.


Add Delivery Methods
A [Delivery Method is where Gateway would send transactions to](https://gateway.sanctum.so/docs/delivery-methods). You can add multiple delivery methods to get better transaction landing rates.Here’s how to set up a delivery method:
1
Navigate to Dashboard > Delivery Methods
This page shows you all the methods created and linked to the current project![Screenshot 2025-06-25 at 1.20.48 PM.png](https://mintcdn.com/sanctum-8b4c5bf5/aA2NSy1MLgkLh8kE/images/Screenshot2025-06-25at1.20.48PM.png?fit=max&auto=format&n=aA2NSy1MLgkLh8kE&q=85&s=fa7a6d1250b635fad29188bb58f93897)
2
Create a Delivery Method
In the dialog, select the type of delivery method, cluster and add it’s name and endpoint URL![Screenshot 2025-06-25 at 1.21.45 PM.png](https://mintcdn.com/sanctum-8b4c5bf5/aA2NSy1MLgkLh8kE/images/Screenshot2025-06-25at1.21.45PM.png?fit=max&auto=format&n=aA2NSy1MLgkLh8kE&q=85&s=add90bd80b02e2e70621d5d6d2a6d675)
3
Add to Project
Once created, click on “Add to Project” to link it to the current project![Screenshot 2025-06-25 at 1.23.10 PM.png](https://mintcdn.com/sanctum-8b4c5bf5/aA2NSy1MLgkLh8kE/images/Screenshot2025-06-25at1.23.10PM.png?fit=max&auto=format&n=aA2NSy1MLgkLh8kE&q=85&s=4e41a30a9e3d2652a796ac0991653475)
Click [here](https://gateway.sanctum.so/dashboard/delivery-methods) to quickly navigate to this page
Copy API Key
API Keys allows you to authenticate and connect your code to Gateway. A default API key has already been created for your project.Here’s how you can get an API Key:
1
Navigate to Dashboard > API Keys
This page shows you all the API keys created for the project. You can copy the default key or add additional keys as shown below![Screenshot 2025-07-10 at 12.47.23 PM.png](https://mintcdn.com/sanctum-8b4c5bf5/aA2NSy1MLgkLh8kE/images/Screenshot2025-07-10at12.47.23PM.png?fit=max&auto=format&n=aA2NSy1MLgkLh8kE&q=85&s=14e202548bb774c3e31ce333f00aad6e)
2
Create an additional API Key
In the dialog, add the name of your API key. You can create multiple API keys to be used in different environments![Screenshot 2025-06-25 at 1.28.10 PM.png](https://mintcdn.com/sanctum-8b4c5bf5/aA2NSy1MLgkLh8kE/images/Screenshot2025-06-25at1.28.10PM.png?fit=max&auto=format&n=aA2NSy1MLgkLh8kE&q=85&s=6667d296985f1c03a929a5e2641f4ebc)
Click [here](https://gateway.sanctum.so/dashboard/settings/api-keys) to quickly navigate to this page
Integration
Now that you’ve set up your Project and API Key, you can authenticate requests to Gateway by supplying the cluster and the key on the query parameters.Click [here](https://gateway.sanctum.so/docs/endpoints) to learn more about the endpoints and how to use them.
